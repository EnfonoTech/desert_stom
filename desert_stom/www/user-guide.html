1. Overview & Workflow

The Asafat Tailoring app adds a complete tailoring workflow on top of ERPNext. The typical order flow is:

  1. Customer visits → Create / select a Customer
  2. New Sales Order → Select cloth items, set quantities, rates
  3. Take Measurement → Record body measurements for each cloth item
  4. Collect Advance → Record advance payment, order moves to Job Order
  5. Stitching Process → Move through Processing → Process Completed → Ready for Delivery
  6. Complete Order → Create Invoice, collect balance payment, deliver
Tip: The stitching status is shown as a colored indicator on the Sales Order. You can also see it on the Sales Order list view with color-coded rows.

2. Workspace (Home Menu)

Navigate to the Asafat Tailoring workspace from the sidebar. It provides:

  • Quick Access shortcuts: Sales Order, Tailoring Measurement, Customer, Daily Sales Collection report, Sales Order Profit Status report
  • Tailoring section: Link to Tailoring Measurement
  • Selling section: Sales Order, Sales Invoice, Payment Entry, Customer, Item
  • Reports section: Daily Sales Collection, Sales Order Profit Status

3. Creating a Customer

  1. Go to Customer > + Add Customer
  2. Enter Customer Name (required)
  3. Enter Phone number in the custom Phone field — this is used for WhatsApp notifications and search
  4. Save
Tip: The phone number is searchable. When creating a Sales Order, you can type the customer's phone number in the Customer field to find them quickly.

4. Creating a Sales Order

  1. Go to Sales Order > + Add Sales Order
  2. Select Customer — the phone number auto-fills from the customer record
  3. In the Items table:
    • Select the Item Code (cloth type)
    • Set Quantity
    • Set Rate (stitching charge)
    • The UOM dropdown only shows UOMs defined in the Item master (e.g., Meter, M, L)
  4. Set Delivery Date
  5. Submit the Sales Order

After submission, the order starts with stitching status Order.

Tip: The Profit & Loss section at the bottom automatically calculates item cost (from valuation rate), stitching cost (35 SAR per qty), and estimated profit.

5. Taking Measurements

  1. Open the submitted Sales Order
  2. Click Measurements > Add Measurement
  3. A dialog appears showing all items from the SO — check/uncheck items to include
  4. If the customer has previous measurements, check Copy from latest measurement to pre-fill values
  5. Click Create Measurement
  6. The Tailoring Measurement form opens with:
    • Customer info and items pre-populated
    • A thobe diagram showing measurement points
    • A progress bar tracking form completion
  7. Fill in the body measurements:
    • Length, Shoulder, Sleeve Length (required)
    • Loose 1, Loose 2, Bottom (required)
    • Bottom Size, Sleeve Loose, Shoulder Alt, Sleeve Alt (optional)
  8. Select design options using visual popup selectors:
    • Collar Style — Shape, Saada, Francy, Malaki, None
    • Special Button — Normal, Special, Gafit, No Button
    • Thobe model — Saudi, Gulf, Kuwait
  9. Submit the measurement → the SO status automatically moves to Measurement

6. Multi-Item Measurements

When a Sales Order has multiple cloth items (e.g., Saudi Thobe + Gulf Thobe), each item gets its own set of measurements:

  1. The Cloth Name dropdown at the top shows all items
  2. Select an item to enter its measurements
  3. Switch to the next item using the dropdown
  4. An indicator shows "Editing: Item Name (1 of 2)"
  5. For the 2nd item onwards, a "Copy from [First Item]" button appears — click it to copy measurements from the first item, then adjust only what differs
  6. All measurements are stored per-item and persist independently
Tip: If the same item appears multiple times in the SO (e.g., 2x Saudi Thobe), they appear as "Saudi Thobe #1" and "Saudi Thobe #2" in the dropdown.

7. Collecting Advance Payment

  1. Open the submitted Sales Order (status must be Order or Measurement)
  2. Click Payments > Collect Advance
  3. Enter:
    • Amount
    • Mode of Payment (Cash, Bank Transfer, etc.)
    • Reference No (optional)
  4. Click Create Payment

A Payment Entry is created and linked to the Sales Order. The Advance Collected and Outstanding Amount fields update automatically.

After the first advance payment, the stitching status moves to Job Order and a WhatsApp message link opens for the customer.

8. Stitching Status Workflow

Each Sales Order progresses through these statuses:

StatusColorHow it changes
OrderBlueAutomatic — when SO is submitted
MeasurementPurpleAutomatic — when measurement is submitted
Job OrderOrangeAutomatic — after first advance payment
ProcessingYellowManual — click Status > Processing
Process CompletedCyanManual — click Status > Process Completed
Ready for DeliveryGreenManual — click Status > Ready for Delivery (sends WhatsApp)
DeliveredGreyAutomatic — after Complete Order
WhatsApp: When status changes to Ready for Delivery, a WhatsApp message link opens automatically so you can notify the customer that their garment is ready.

9. Completing an Order

When the order is Ready for Delivery, the Complete Order button appears:

  1. Click Complete Order
  2. The dialog shows:
    • Billing Mode — Full Billing, Invoice Only, Delivery Only, Payment Only
    • Items — check/uncheck items to include
    • Discount — None, Percentage, or Amount
    • Payment — total, advance paid, and amount to collect now
    • Document Options — toggle Sales Invoice, Payment Entry, Delivery Note creation
    • Print After Create — automatically open print for Invoice and/or Delivery Note
  3. Click Complete

The system creates the selected documents (Invoice, Payment Entry, Delivery Note), marks the SO as Delivered, and optionally opens print views.

10. Reports

Daily Sales Collection

Shows all sales invoices and payment entries for a date range:

  • Filters: From Date, To Date, Company, Mode of Payment
  • Columns: Date, Invoice, Customer, Sales Order, Invoice Amount, Payment Entry, Mode of Payment, Collected Amount, Outstanding
  • Invoices with linked payments appear together; advance/standalone payments appear separately

Sales Order Profit Status

Shows profit analysis for each Sales Order:

  • Filters: From Date, To Date, Company, Stitching Status, Customer
  • Columns: Sales Order, Date, Customer, Stitching Status, Grand Total, Item Cost, Stitching Cost (35 SAR/qty), Total Cost, Estimated Profit, Advance Collected, Outstanding, Measurement Count
  • Profit is shown in green (positive) or red (negative)
  • Stitching status is color-coded to match the workflow

11. Searching by Phone Number

You can search for customers and orders using phone numbers:

  • In the Customer link field (on Sales Order or anywhere), type the phone number to find the customer
  • In the Sales Order list, use the search bar to search by phone number
  • The phone number is displayed prominently on the Sales Order form in blue

12. Print Formats

  • Measurement Card — compact print of all measurements for a Tailoring Measurement
  • Tailoring Invoice — custom Sales Invoice print format
  • Delivery Slip — custom Delivery Note print format

Print formats can be selected when completing an order using the Print After Create option.
